This is a release candidate build for 1.2.1 with no known bugs or missing features.

This build will install on parallel over the App Store version of ZotPad.

The previous build had the support features disabled, but they are enabled now.

----- Changes since 1.2  -----

- The fields in item detail view now wrap to multiple lines instead of being truncated and support copy paste and tapping on links
- Log view is initially scrolled to bottom
- Fixes a crash that was caused by WebDAV files that had zero lenght
- Adds additional error checking to block a couple of crashes
